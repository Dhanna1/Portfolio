PUBLISHING WITH GITHUB PAGES

1. Create a free GitHub account.
2. Create a new PUBLIC repository named portfolio.
3. Extract Dalton_Hanna_Portfolio_V4.zip.
4. Upload everything INSIDE the extracted Dalton_Hanna_Portfolio_V4 folder:
   index.html
   projects/
   assets/
   downloads/
   README.txt
5. Commit the upload.
6. Open the repository's Settings.
7. Select Pages.
8. Under Build and deployment:
   Source: Deploy from a branch
   Branch: main
   Folder: / (root)
9. Save and wait several minutes.
10. Your public URL will normally be:
    https://YOUR-USERNAME.github.io/portfolio/

SHARING
Place the public URL on your resume, LinkedIn Featured section, GitHub profile,
email signature, job applications and interview thank-you emails.

CUSTOM DOMAIN
After the site is published, a domain such as daltonhanna.com can be connected
through the GitHub Pages settings and your domain provider's DNS settings.
